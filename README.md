# ![Logo](chrome/app/theme/chromium/product_logo_64.png) Chromosomium

Chromosomium is an open-source browser project that aims to build a safer, faster,
and more stable way for all users to experience the chromosoms.

The project's web site is https://www.chromosomium.org.

Documentation in the source is rooted in [docs/README.md](docs/README.md).
